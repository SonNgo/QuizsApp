How to run:

1. git clone git@github.com:SonNgo/QuizsApp.git
2. cd QuizsApp
3. cd server-api 
4. bundle install 
5. rails s (run server on localhost:3000)
6. Ctrl + Shift + T (open new terminal)
7. cd ../client
8. python -m SimpleHTTPServer (run client on localhost:8000)
9. open http://localhost:8000/
